# Plan 07-01: OG SVG Generation

## Objective

Write `scripts/generate-og-svgs.mjs` that reads `stops.json` and generates one SVG per stop (era color, title, scientist, year badge, branding). Run the script and commit all 50 SVG files.

## Tasks

### Task 1: Create scripts directory and generator script

<files>
  - scripts/generate-og-svgs.mjs (create)
</files>

<action>
Write a Node.js ESM script at `scripts/generate-og-svgs.mjs` that:
1. Reads `Episodio4/assets/data/stops.json`
2. For each stop, generates a 1200x630 SVG (standard OG image size)
3. SVG design:
   - Dark background (#0d0d0f — close to var(--color-void))
   - Era accent color band/border on left edge (8px wide)
   - Era label badge in top-left
   - Site branding "How Physics Works" in top-right (small, muted)
   - Stop number (zero-padded, e.g., "01") centered-left, large, muted
   - Stop title in Cormorant Garamond-like large serif (white)
   - Scientist name below title (smaller, era-colored)
   - Year badge bottom-right (rounded rect, era color bg)
   - Subtle horizontal rule separator
4. Era color hex values (approximations of oklch colors):
   - ancient: #c4922a (warm amber)
   - revolution: #4ca86b (sage green)
   - classical: #5b8fd4 (steel blue)
   - modern: #b06fd0 (purple)
   - contemporary: #d45b5b (coral red)
5. Saves each SVG to `Episodio4/stops/{stop-id}/og-image.svg`
6. Prints a summary line per file written
</action>
