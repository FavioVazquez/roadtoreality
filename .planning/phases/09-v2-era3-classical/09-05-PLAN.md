# Plan 09-05: Stops 023 + 024 — The Doppler Effect and Boltzmann Entropy

## Objective

Implement two full simulations: (1) the Doppler effect with animated growing wavefront circles emitted from a moving source and live frequency readouts for ahead/behind observers; (2) Boltzmann entropy — a split-canvas with 60 bouncing elastic-collision particles and a live Maxwell-Boltzmann histogram (left panel) alongside a partition-wall removal experiment with a rising entropy meter (right panel). Stop 024 is a ~600-line sim.

## Context

Both stops are `isStub: true`. Reference: stop-014 sim.js (ES5 IIFE, split-canvas, `splitX` in `resize()`).

**Stop 023 key constraint (RESEARCH.md Pitfall 7):** Cap source speed at 0.95 × v_wave to stay subsonic. At supersonic speeds, forward wavefronts collapse to zero apparent spacing and become invisible.

**Stop 024 key constraints:**
- Pitfall 2: Use spatial grid OR cap at 60 particles with squared-distance (no Math.sqrt) for collision detection. With 60 particles in 10×10 cells, each cell averages 0.6 particles — only check same-cell + adjacent cells.
- Pitfall 3: `splitX = W * 0.50` computed inside `resize()`, not at setup time.
- Pitfall 9: Histogram bars use exponential moving average: `displayH[i] = displayH[i] * 0.85 + actualH[i] * 0.15`. Prevents flicker.
- Partition wall removal is the key interaction — animate the wall sliding upward (y decreases each frame until wall disappears). Entropy meter climbs as particles spread into right half.

**ES5 constraint:** All `var`, named functions, string concatenation. No arrow functions, no const/let, no template literals.

Era color: `rgba(82,133,200,0.8)`.

Wall collisions (left panel bounds: x in [0, splitX], y in [0, H]):
- If p.x - p.radius < 0: vx = Math.abs(vx)
- If p.x + p.radius > splitX: vx = -Math.abs(vx)
- If p.y - p.radius < 0: vy = Math.abs(vy)
- If p.y + p.radius > H: vy = -Math.abs(vy)

Particle-particle elastic collisions (cap at 60 particles, use squared-distance without Math.sqrt for threshold test):
```
for (var i = 0; i < gasParticles.length; i++) {
  for (var j = i+1; j < gasParticles.length; j++) {
    var dx = gasParticles[j].x - gasParticles[i].x;
    var dy = gasParticles[j].y - gasParticles[i].y;
    var dist2 = dx*dx + dy*dy;
    var minDist = gasParticles[i].radius + gasParticles[j].radius;
    if (dist2 < minDist * minDist) {
      // Elastic collision: exchange velocity components along collision axis
      var dist = Math.sqrt(dist2); // Only compute sqrt when collision detected
      var nx = dx/dist; var ny = dy/dist;
      var dvx = gasParticles[j].vx - gasParticles[i].vx;
      var dvy = gasParticles[j].vy - gasParticles[i].vy;
      var dot = dvx*nx + dvy*ny;
      if (dot < 0) { // approaching
        gasParticles[i].vx += dot*nx; gasParticles[i].vy += dot*ny;
        gasParticles[j].vx -= dot*nx; gasParticles[j].vy -= dot*ny;
        // Separate overlapping particles
        var overlap = minDist - dist;
        gasParticles[i].x -= nx*overlap*0.5; gasParticles[i].y -= ny*overlap*0.5;
        gasParticles[j].x += nx*overlap*0.5; gasParticles[j].y += ny*overlap*0.5;
      }
    }
  }
}
```
With 60 particles: 60*59/2 = 1770 pairs checked per frame. At 60fps = 106,200 checks/second. Manageable on modern hardware without spatial grid (per RESEARCH.md allowance for <=60 particles).

Temperature slider effect: when `temperature` changes, scale all particle velocities: `scale = Math.sqrt(newTemp / oldTemp); p.vx *= scale; p.vy *= scale;` Apply to all gasParticles.

- Each frame: compute actualH[i] = count of particles with speed in bin i, normalized to max bin count.
- Smooth: `displayH[i] = displayH[i] * 0.85 + actualH[i] * 0.15`
- Draw bars: inside left panel, at bottom section (y from H*0.7 to H*0.95). Bar width = (splitX - 20) / 10. Bar height = displayH[i] * (H*0.22).
- Color: era blue `rgba(82,133,200,0.7)`.

- Label: "Speed distribution" at top of histogram area.

**Right panel — entropy partition experiment (x: splitX to W, y: 0 to H):**

Wall collision bounds for right panel:
- Left wall at x = splitX; right wall at x = W; top at y = 0; bottom at y = H.
- When `partitionActive = true`: vertical partition wall at x = splitX + (W-splitX)*0.5. Particles in left-right-panel-half bounce off this wall too. Right half of right panel is empty.
