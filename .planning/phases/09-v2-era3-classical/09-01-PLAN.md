## Context

Both stops are currently `isStub: true` with placeholder sim.js files. The reference implementation is `Episodio4/stops/014-hooke-elasticity/sim.js` (503 lines, ES5 IIFE). Key patterns:
- `window.SimAPI = { start, pause, reset, destroy }` assigned synchronously at IIFE body level
- Canvas DPR scaling: `canvas.width = Math.round(W * dpr); ctx.setTransform(dpr, 0, 0, dpr, 0, 0)`
- All geometry recalculated inside `resize()` callback, not at setup time
- Animation loop: `function loop() { if (!running) return; t += 0.016; draw(); raf = requestAnimationFrame(loop); }`
- ES5 only: `var`, named `function` declarations, string concatenation (`'string ' + val`)
- Era color for Classical Physics: use `rgba(82,133,200,0.8)` (`#5283c8`) for canvas drawing
- `prefers-reduced-motion` guard: check before `requestAnimationFrame`, draw static frame if reduced
- Controls declared in HTML; wired with `getElementById` + `addEventListener('input', ...)` in `setup()`

Write a complete ES5 IIFE sim following the stop-014 pattern.

**Physics model:**
- Three rigid bodies, same mass M=1 kg, same radius/half-length R=0.5 (normalized), different moments of inertia:
  - Disk: I_disk = 0.5 * M * R^2 = 0.125
  - Rod (thin, rotating about center): I_rod = (1/12) * M * (2R)^2 = 0.0833
  - Ring: I_ring = M * R^2 = 0.25
- Constant torque tau applied to all three (tau slider: 0.1–1.0, default 0.4).
- Angular acceleration: alpha = tau / I. Each body accumulates omega += alpha * dt each frame (dt = 0.016).
- Cap omega at 20 rad/s for display purposes. Reset omega = 0 on Reset.

**Canvas layout:**
- Three equal horizontal thirds: left = Disk, center = Rod, right = Ring.
- Draw each body rotated by its accumulated angle theta (theta += omega * dt each frame):
  - Disk: filled circle with a diameter line at angle theta to show rotation. Color the quadrants differently (two halves: front half `rgba(82,133,200,0.6)`, back half `rgba(82,133,200,0.25)`). Draw a radius mark (short line from center to edge) at angle theta.
  - Rod: a rectangle (8px wide, 80px long) centered in its third, rotated by theta. Color `rgba(82,133,200,0.8)`.
  - Ring: an annulus (outer radius ~40px, inner radius ~30px), rotated indicator line from center to outer edge at theta. Color `rgba(82,133,200,0.8)` for the ring outline, fill transparent.
- Label each body: "Disk  I=½MR²", "Rod  I=¹⁄₁₂ML²", "Ring  I=MR²" below its drawing.
- Angular velocity graph: a small graph at canvas bottom spanning full width. x-axis = time (rolling window, last 4 seconds). y-axis = omega (0–20 rad/s). Three lines: Disk (blue), Rod (green `#61bd67`), Ring (gold `rgba(200,160,50,0.8)`). The diverging lines are the visual payoff — same torque, fastest omega for rod (smallest I), slowest for ring (largest I).
- Keep a circular buffer of last 250 omega samples per body for the graph.

**Controls wired in setup():**
1. `torque-slider` (range 0.1–1.0, step 0.01, default 0.4): updates `tau`, triggers recalculation of alpha values.
2. `mass-slider` (range 0.5–3.0, step 0.1, default 1.0): updates M (affects all I values proportionally); recalculate I and alpha. This is the second meaningful control.

**SimAPI:** start/pause/reset/destroy. Reset: set omega=0 for all three bodies, theta=0, clear graph buffers, redraw.

**ES5 constraint:** All `var`, named functions, string concatenation.
