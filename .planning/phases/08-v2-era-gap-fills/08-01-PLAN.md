## Context

**Stop-shell.js integration:** `stop-shell.js` line 90 checks `wrapper.dataset.autoplay !== 'false'`. Because this stop must be silent until a user gesture, the attribute must be the string `"false"`. The IntersectionObserver will still call `SimAPI.start()` — that is fine; the canvas wave animation may start, but no audio will play until a button is clicked.

**AudioContext lazy creation:** Do NOT create `AudioContext` at IIFE startup or in `SimAPI.start()`. Create it inside the first ratio button `click` handler. The button click is a valid user gesture; `IntersectionObserver` is not. After creating it, always call `audioCtx.resume()` before scheduling nodes.

**One OscillatorNode per press:** Create a fresh `OscillatorNode` + `GainNode` on each button click. Connect them, call `osc.start()`, schedule `osc.stop(now + 0.85)`, then discard. Do NOT reuse stopped oscillators.

**Envelope — no clicks:** Use `gain.gain.setValueAtTime(0, now)`, `linearRampToValueAtTime(0.5, now + 0.01)`, `exponentialRampToValueAtTime(0.0001, now + 0.81)`. The target for `exponentialRampToValueAtTime` must never be exactly `0` — use `0.0001`.

**Standing wave formula:**
```
y_offset = A * sin(n * PI * (x - x0) / L) * cos(omega_anim * t)
```
where `n` is the harmonic number, `x0` is the string left endpoint, `L` is the string length in canvas pixels, `omega_anim` is an aesthetic animation speed (not audio frequency), and `t` is the animation clock.

**Ratio → harmonic n mapping:**
- 1:1 (unison) → n=1, freq = 220 Hz
- 2:1 (octave) → n=2, freq = 440 Hz
- 3:2 (perfect fifth) → n=3, freq = 330 Hz
- 4:3 (perfect fourth) → n=4, freq = 293 Hz (220 * 4/3)
- 5:4 (major third) → n=5, freq = 275 Hz (220 * 5/4)
- 9:8 (whole tone) → n=9, freq = 247.5 Hz (220 * 9/8)

**DPR-aware canvas:** Follow the pattern from stops 010 and 012. Read `window.devicePixelRatio || 1`, multiply `canvas.width`/`canvas.height` by DPR, set `.style.width`/`.style.height` in CSS px, call `ctx.setTransform(dpr, 0, 0, dpr, 0, 0)`.

**ES5 IIFE pattern:** Use `var` declarations only. No `const`, no `let`, no arrow functions. Pattern: `(function () { 'use strict'; ... }());`

**Reduced motion:** Check `window.matchMedia('(prefers-reduced-motion: reduce)').matches`. If true, render only the static node positions — no animation loop.

**Existing reference sims:** `/Episodio4/stops/010-galileo-pendulum/sim.js` and `/Episodio4/stops/012-newton-laws/sim.js` are the canonical patterns for `setup()`, `DOMContentLoaded`, `SimAPI`, and canvas DPR. Read them before writing sim.js.

**OG images:** Already generated in Phase 07. Do NOT regenerate — no metadata changed.

**Audio function `playTone(freqHz)`:**
```javascript
function playTone(freqHz) {
  if (!audioCtx) { audioCtx = new (window.AudioContext || window.webkitAudioContext)(); }
  if (audioCtx.state === 'suspended') { audioCtx.resume(); }
  var osc = audioCtx.createOscillator();
  var gain = audioCtx.createGain();
  osc.connect(gain);
  gain.connect(audioCtx.destination);
  osc.type = 'sine';
  osc.frequency.value = freqHz;
  var now = audioCtx.currentTime;
  gain.gain.setValueAtTime(0, now);
  gain.gain.linearRampToValueAtTime(0.5, now + 0.01);
  gain.gain.exponentialRampToValueAtTime(0.0001, now + 0.81);
  osc.start(now);
  osc.stop(now + 0.85);
}
```

```javascript
window.SimAPI = {
  start: function () {
    if (running) return;
    running = true;
    var btn = document.getElementById('sim-play-btn');
    if (btn) { btn.dataset.state = 'playing'; btn.innerHTML = '&#9646;&#9646; Pause'; }
    if (!reduced) loop();
  },
  pause: function () {
    running = false;
    if (raf) { cancelAnimationFrame(raf); raf = null; }
    var btn = document.getElementById('sim-play-btn');
    if (btn) { btn.dataset.state = 'paused'; btn.innerHTML = '&#9654; Play'; }
  },
  reset: function () {
    t = 0;
    currentN = 1;
    var btns = document.querySelectorAll('[data-ratio]');
    for (var i = 0; i < btns.length; i++) { btns[i].removeAttribute('data-state'); }
    if (canvas && ctx) draw();
  },
  destroy: function () {
    window.SimAPI.pause();
    if (audioCtx) { audioCtx.close(); audioCtx = null; }
    if (canvas && canvas.parentNode) canvas.parentNode.removeChild(canvas);
  }
};
```
