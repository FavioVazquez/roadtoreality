## Objective

## Context

**Current state:** `sim.js` is an oscillating teaser that exposes `window.SimAPI`. `index.html` has `data-autoplay="true"` (correct — keep it). `stops.json` has `isStub: true`.

**data-autoplay is already "true":** Stop 014 is designed to animate on load. Do NOT change `data-autoplay`. The `stop-shell.js` integration calls `SimAPI.start()` on intersection, which starts the spring oscillation with the default weight.

**Single-canvas split layout:** Divide the canvas into two regions using a vertical midpoint. Left half: spring visual. Right half: F vs. x graph. Use `var splitX = W * 0.5;` as the dividing line. This matches the dual-panel approach used in stops 010 and 012.

**Material presets:**
```
Steel:  k=500 N/m, yieldX=0.04 m, ruptureX=0.06 m  (brittle: tiny plastic zone)
Rubber: k=50  N/m, yieldX=0.40 m, ruptureX=1.20 m  (elastic: large plastic zone)
Glass:  k=800 N/m, yieldX=0.01 m, ruptureX=0.015 m (instant snap: almost no plastic zone)
```

**Piecewise spring model (slider-driven, no ODE):**
The slider value is a 0–1 fraction of the rupture displacement for the selected material. Extension is computed directly from slider position, not from simulated forces:
- Elastic zone (`sliderFrac <= yieldFrac`): `x = sliderFrac * ruptureX`; force `F = k * x`; coils uniform.
- Plastic zone (`yieldFrac < sliderFrac < 1.0`): `x = yieldX + (sliderFrac - yieldFrac)^0.4 * (ruptureX - yieldX) / (1 - yieldFrac)^0.4`; force `F = k * yieldX + k_plastic * (x - yieldX)` where `k_plastic = k * 0.1`; coils drawn with non-uniform spacing (outer coils wider).
- Rupture (`sliderFrac >= 1.0`, i.e., slider at max): trigger snap once; set `ruptured = true`; disable slider.

**Ruptured boolean state:** Maintain a `var ruptured = false;` top-level variable. On `ruptured = true`: trigger the snap animation (see below), stop further slider processing, show "RUPTURED" label. On material change: call `reset()`, which sets `ruptured = false` and re-enables slider.

**Snap animation:** On rupture, run a 300 ms canvas shake effect — offset `ctx.translate` by a random small amount each frame for 300 ms, then restore. Additionally flash the canvas background briefly (fill with red at alpha 0.3, then fade). Use a `snapTimer` counter (frames) set to 18 (at ~60fps = 300 ms); decrements each frame until 0.

**F vs. x graph (right panel):**
- Axes: x axis = extension (m) from 0 to `ruptureX`; y axis = force (N) from 0 to `k * ruptureX`.
- Draw the full material curve first as background:
  - Elastic segment: green line (`#61bd67` = revolution era green) from (0,0) to (yieldX, k*yieldX).
  - Plastic segment: muted gray line from (yieldX, k*yieldX) curving to (ruptureX, F_rupture).
  - Rupture dot: red filled circle at (ruptureX, F_rupture).
- Draw a live tracking dot (white, radius 5) at the current (x, F) position, clamped to the curve.
- Label axes: "x (m)" at bottom-right, "F (N)" at top-left. Use small canvas font.

**Spring coil visual (left panel):**
- Spring anchored at top-center of left panel.
- Natural spring height: ~35% of panel height.
- Extended spring height: natural height + extension scaled to canvas (cap visual extension at 80% of panel height minus anchor).
- Weight block (rectangle) hangs below spring; label shows mass in kg if desired (F = mg, so m = F / 9.81 — display as "~X kg").
- In plastic zone: coils drawn non-uniformly — outer coils (top and bottom thirds) wider than center coils. Use a sine-modulated zigzag amplitude: `coilW * (1 + 0.5 * Math.sin(i * Math.PI / coils))`.
- In ruptured state: draw broken spring ends (two short stubs) and weight block falling (animate weight block Y position increasing until off-canvas).

**autoplay default state:** Weight slider defaults to 30% of max (just inside elastic zone). On `SimAPI.start()`, the sim draws this initial state.

**DPR-aware canvas:** Same pattern as Stop 002 and stops 010/012. `dpr = window.devicePixelRatio || 1`. Multiply canvas width/height by dpr, set style to CSS px, `ctx.setTransform(dpr, 0, 0, dpr, 0, 0)`.

**ES5 IIFE pattern:** `var` only. No `const`, `let`, arrow functions. Pattern: `(function () { 'use strict'; ... }());`

**Reduced motion:** If `window.matchMedia('(prefers-reduced-motion: reduce)').matches`, draw the static spring at default position without animation loop.

**Existing reference sims:** `/Episodio4/stops/010-galileo-pendulum/sim.js` and `/Episodio4/stops/012-newton-laws/sim.js` are canonical patterns. Read them before writing sim.js.

**OG images:** Already generated in Phase 07. Do NOT regenerate.

**`getMaterial()` helper:** returns `MATERIALS[currentMaterial]`.

**`getState(frac)` function:** Given `frac` (0–1), material `m`, returns `{ x, F, zone }` where zone is `'elastic'`, `'plastic'`, or `'ruptured'`:
- `yieldFrac = m.yieldX / m.ruptureX`
- If `frac <= yieldFrac`: `x = frac * m.ruptureX; F = m.k * x; zone = 'elastic'`
- Else if `frac < 1`: `t_p = (frac - yieldFrac) / (1 - yieldFrac); x = m.yieldX + Math.pow(t_p, 0.4) * (m.ruptureX - m.yieldX); F = m.k * m.yieldX + m.k * 0.1 * (x - m.yieldX); zone = 'plastic'`
- Else: `x = m.ruptureX; F = m.k * m.yieldX; zone = 'ruptured'`

**`drawSpring(ctx, anchorX, anchorY, endY, zone, coils)` function:**
- `coils` = 12.
- Draw vertical zigzag from `(anchorX, anchorY)` to `(anchorX, endY)`.
- `step = (endY - anchorY) / (coils * 2 + 2)`.
- For elastic zone: `coilW = 16` px (uniform).
- For plastic zone: modulate width: `coilW = 16 * (1 + 0.5 * Math.abs(Math.sin(i * Math.PI / coils)))`.
- Color: `#61bd67` for elastic, muted gray `#7a7a7a` for plastic, red for ruptured stubs.

**`drawGraph(ctx, m, currentX, currentF, ruptured)` function:**
- Graph area: `gx0 = splitX + W*0.05, gy0 = H*0.1, gx1 = W*0.95, gy1 = H*0.88`.
- Map x-values to canvas: `px(x) = gx0 + (x / m.ruptureX) * (gx1 - gx0)`.
- Map F-values to canvas: `py(F) = gy1 - (F / (m.k * m.ruptureX)) * (gy1 - gy0)`.
- Draw axes (muted gray lines).
- Draw axis labels: "x (m)" below x-axis at right, "F (N)" left of y-axis at top.
- Draw elastic segment: green stroke from (0,0) to (m.yieldX, m.k * m.yieldX).
- Draw plastic segment: gray stroke from yield point to rupture point (use 20 intermediate points).
- Draw rupture dot: red filled circle at (m.ruptureX, m.k * m.yieldX), radius 5.
- Draw live tracking dot: white filled circle at (currentX, currentF), radius 5 — clamp to curve, hidden if ruptured.

**`draw()` function:**
- Clear canvas.
- If `snapTimer > 0`: apply random translate offset (`ctx.save(); ctx.translate(random -3..3, random -3..3)`) for shake; also draw a red-tinted overlay (`ctx.fillStyle = 'rgba(200,50,50,0.15)'; ctx.fillRect...`); decrement `snapTimer`.

- If ruptured: draw broken stubs; advance `weightY += 4` (weight falls).
- Else: draw spring; draw weight block below springEndY; draw mass label.
- Draw dividing line between panels (subtle vertical separator).
- Draw F vs. x graph in right panel.
- If snapTimer > 0: `ctx.restore()` to end shake transform.

**`window.SimAPI`** set at IIFE body level:
```javascript
window.SimAPI = {
  start: function () {
    if (running) return;
    running = true;
    var btn = document.getElementById('sim-play-btn');
    if (btn) { btn.dataset.state = 'playing'; btn.innerHTML = '&#9646;&#9646; Pause'; }
    if (!reduced) loop();
    else draw();
  },
  pause: function () {
    running = false;
    if (raf) { cancelAnimationFrame(raf); raf = null; }
    var btn = document.getElementById('sim-play-btn');
    if (btn) { btn.dataset.state = 'paused'; btn.innerHTML = '&#9654; Play'; }
  },
  reset: function () {
    ruptured = false;
    snapTimer = 0;
    weightFrac = 0.3;
    weightY = 0;
    var slider = document.getElementById('weight-slider');
    if (slider) { slider.value = 0.3; slider.disabled = false; }
    draw();
  },
  destroy: function () {
    window.SimAPI.pause();
    if (canvas && canvas.parentNode) canvas.parentNode.removeChild(canvas);
  }
};
```

When `ruptured` becomes true inside `draw()`, also disable the slider: `var slider = document.getElementById('weight-slider'); if (slider) slider.disabled = true;`
